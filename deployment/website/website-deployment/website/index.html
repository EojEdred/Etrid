ASF Consensus

Asynchronous-Synchronous Finality Protocol

Overview

ASF (Asynchronous-Synchronous Finality) is √ãTRID's revolutionary consensus mechanism that combines the speed of asynchronous block production with the security of synchronous finality. This hybrid approach delivers sub-second transaction finality while maintaining 171,000+ TPS throughput across the network.

Key Features

Allows rapid block production with probabilistic confidence before full finality

Lightning-Bloc Layer 2

High-Speed Off-Chain Scaling Solution

Overview

Lightning-Bloc is √ãTRID's Layer 2 scaling solution that enables ultra-fast, low-cost transactions through off-chain state channels. By batching transactions off-chain and periodically settling on FlareChain, Lightning-Bloc achieves over 1 million transactions per second with near-instant finality and minimal fees.

Key Features

Multi-party payment channels for instant peer-to-peer transactions

Instant Finality
