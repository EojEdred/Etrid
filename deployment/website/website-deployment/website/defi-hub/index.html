Configuration Location

Edit the validator configuration file at:

/Users/macbook/Desktop/etrid/infrastructure/config/validator-ips.json

Configuration Structure

{
  "validators": [
    {
      "id": 1,
      "name": "Validator Name",
      "region": "Azure West US",
      "role": "Director/Bootstrap",
      "ip": "20.xxx.xxx.xxx",
      "sshUser": "username",
      "accessible": true
    }
  ]
}

Field Descriptions

id

Unique identifier for the validator (number)

name

Display name for the validator

region

Geographic location or cloud region

role

Validator role (e.g., Director, Developer, Network Node)

ip

IP address for SSH connection

sshUser

SSH username for remote access

accessible

Whether SSH access is enabled (true/false)

Quick Commands

# Edit configuration
nano infrastructure/config/validator-ips.json
# Validate JSON syntax
cat infrastructure/config/validator-ips.json | jq .
# Restart monitoring
cd pinokio && npm run validator:monitor
