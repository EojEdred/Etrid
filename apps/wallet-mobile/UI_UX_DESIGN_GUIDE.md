## ğŸ¨ Design Philosophy

### Core Principles

1. **Banking Familiarity**: Use terms people know (checking, savings, deposit, withdraw)
2. **Progressive Disclosure**: Show simple by default, advanced options hidden
3. **Zero Blockchain Knowledge**: No addresses, hashes, gas fees in main UI
4. **Instant Feedback**: Optimistic UI updates, background confirmation
5. **Visual Hierarchy**: Most important info biggest, secondary info smaller
6. **Accessibility**: WCAG 2.1 AA compliance, screen reader support
