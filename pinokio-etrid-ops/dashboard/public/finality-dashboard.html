üí° Understanding ASF (Asynchronous Safe Finality)

  • Finality Time: Time taken for a block to become irreversibly finalized on the blockchain
  • Target Finality: Etrid aims for finality times under 20 seconds for optimal performance
  • Block Time: Average time between block production (~5 seconds target)
  • Block Lag: Difference between the latest produced block and the finalized block
