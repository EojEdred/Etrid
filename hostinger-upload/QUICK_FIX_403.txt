QUICK FIX FOR 403 FORBIDDEN ERROR - ËTRID HOSTINGER UPLOAD
=================================================================

🚨 MOST COMMON CAUSE: FILE PERMISSIONS

-------------------------
SOLUTION (Do This First):
-------------------------

1. Log into Hostinger File Manager

2. Navigate to subdomain folder:
   Example: /public_html/validator/

3. Select ALL files and folders (click checkbox at top)

4. Right-click → "Permissions" or "Change Permissions"

5. Set these permissions:
   ✅ Folders: 755
   ✅ Files: 644

6. Check: "Apply to subdirectories recursively"

7. Click "Change" or "Apply"

8. Wait 30 seconds

9. Test URL: https://validator.etrid.org

-------------------------
STILL NOT WORKING?
-------------------------

Check file structure:

Your folder should look like this:

/public_html/validator/
├── index.html          ← MUST be here (not in subfolder)
├── .htaccess          ← Included in new zip
├── _next/             ← Assets folder
├── 404.html
└── ...other files

NOT like this:

/public_html/validator/
└── validator/         ← WRONG! Extra nested folder
    ├── index.html
    └── _next/

If you see nested folders:
1. Move all files UP one level
2. Delete empty nested folder

-------------------------
SUBDOMAIN SETTINGS
-------------------------

In Hostinger panel → Domains → Subdomains

Make sure each subdomain points to:

validator.etrid.org    → /public_html/validator
watchtower.etrid.org   → /public_html/watchtower
masterchef.etrid.org   → /public_html/masterchef
wallet.etrid.org       → /public_html/wallet
