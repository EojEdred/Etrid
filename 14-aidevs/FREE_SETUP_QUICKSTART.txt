│ OPTION 1: YOUR MAC (EASIEST - START IN 5 MINUTES)          │
└─────────────────────────────────────────────────────────────┘

1. Install Ollama (local LLM - replaces $100/month Claude)
   brew install ollama
   ollama serve  # In one terminal
   ollama pull llama3.1:8b  # In another terminal

2. Update your generator.py
   cd /Users/macbook/Desktop/etrid/ai-devs/social/content
   cp generator_with_ollama.py generator.py

3. Configure .env
   cd /Users/macbook/Desktop/etrid/ai-devs/social
   nano .env

   Add:
   USE_LOCAL_LLM=true
   LOCAL_LLM_URL=http://localhost:11434
   LOCAL_LLM_MODEL=llama3.1:8b
   BLOCKCHAIN_WS_URL=wss://rpc.etrid.network
   TWITTER_API_KEY=your_key
   TWITTER_API_SECRET=your_secret
   TWITTER_ACCESS_TOKEN=your_token
   TWITTER_ACCESS_SECRET=your_secret

4. Test
   python workflows/daily_stats.py --dry-run

5. Run
   python workflows/daily_stats.py
   python workflows/twitter_mentions.py --mode stream

COST: $0/month
QUALITY: ⭐⭐⭐⭐⭐ (llama3.1 is excellent)

1. Create Oracle Cloud account
   → oracle.com/cloud/free
   → Sign up (free tier, no credit card required after trial)

2. Create VM
   → Compute > Instances > Create Instance
   → Image: Ubuntu 22.04
   → Shape: VM.Standard.A1.Flex (ARM)
   → OCPU: 4, RAM: 24GB
   → This is FREE FOREVER ✨

3. SSH in and install
   ssh ubuntu@<your-vm-ip>

   # Install Ollama
   curl -fsSL https://ollama.com/install.sh | sh
   ollama pull llama3.1:8b

   # Install Python
   sudo apt update
   sudo apt install -y python3.11 python3.11-venv python3-pip git

   # Setup AI Devs (upload your code or git clone)
   cd ~/etrid/ai-devs/social
   python3.11 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

4. Configure (same as Option 1)
   cp .env.example .env
   nano .env
   # Add USE_LOCAL_LLM=true, Twitter keys, etc.

   # Update generator.py
   cp content/generator_with_ollama.py content/generator.py

5. Setup auto-start
   # Create systemd service (see DEPLOY_FREE_OR_CHEAP.md)
   sudo systemctl enable etrid-social
   sudo systemctl start etrid-social
