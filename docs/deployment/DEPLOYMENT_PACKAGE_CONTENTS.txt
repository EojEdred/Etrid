ROLLBACK PROCEDURE
================================================================================

If deployment needs to be reversed:

For each affected VM:
  ssh -i ~/.ssh/etrid_vm1 <vm-address>
  sudo systemctl stop etrid-agent
  sudo systemctl disable etrid-agent
  sudo rm -rf /opt/etrid-agent
  sudo systemctl daemon-reload
  exit

Rollback Time: <10 minutes per VM
Data Loss: None (no data stored on agents)
Impact: Agents stop reporting, Node Exporter continues
Recovery: Re-run deployment script to restore
